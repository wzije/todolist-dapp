# Todolist DApp

This project for learning need

## Prerequisites

1. Node v18.7.0
2. Truffle v5.6.2 (core: 5.6.2)
3. Ganache v7.4.4
4. Solidity - 0.8.17 (solc-js)
5. Node v18.7.0
6. Web3.js v1.7.4
7. VsCode

## Running the application

1. npm install
2. truffle migrate (--reset | optional)
3. npm run start
